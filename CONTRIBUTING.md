## How to Contribute 
1. **Fork the repository** Create your own copy of the repo under your GitHub account.
2. **Create a branch** git checkout yourfirstname/your-feature-name
3. **Make your changes** Keep your code clean and easy to read. When naming, use camel case.
4. **Commit your changes** Write clear commit messages that describe what you did.
5. **Open a Pull Request** Fill out the Pull Request Template.
